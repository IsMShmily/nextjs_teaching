## Next.js 教学
Next.js 是基于 React 的全栈框架，专为构建高性能、SEO 友好的现代 Web 应用设计。

核心优势：

- 服务端渲染 (SSR) 和 静态站点生成 (SSG)：提升加载速度和SEO。

- 内置路由系统：文件即路由，无需手动配置。

- API 路由：轻松创建后端接口。

- TypeScript 支持、CSS Modules、图像优化等开箱即用功能。


## 一、运行

```bash
# 下载依赖
npm i 

# 运行项目
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

打开 [http://localhost:3000](http://localhost:3000)用你的浏览器查看结果。
